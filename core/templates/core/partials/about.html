Studio 314

Version 1.0.0 — Gestion de Clients

Objectif

Studio 314 est une application de gestion de la relation client (CRM) complète, permettant de gérer efficacement votre portefeuille clients avec un suivi détaillé des interactions.

Technologies

  • Django 4.2 Framework Python
  • SQLite Base de données
  • SPA Ajax Frontend
  • CSS3 Interface moderne
  • Vanilla JS Interactivité

Fonctionnalités

  • ✅ CRUD complet des clients
  • ✅ Recherche & filtres avancés
  • ✅ Export CSV
  • ✅ Historique des interactions
  • ✅ Authentification sécurisée
